Nowadays it is more and more popular to use icons in your applications. It is all came from subways. Foreigners who were using subway wasn't able to read words on different boards, for example: exit, don't go this way, stairs etc. That why people started using icons. They explain meening of things without any words.

Modern fonts support a lot of cool icons, however most of the web-developers don't use them because they don't know the correct icon code. Current web-site is made to make it easier to find necessary icon.

If you want to help the project, please share it in your social network, like it in on GitHub or fork it and add new icons on your own. Or even create Pull request on GitHub and I'll be happy to add your changes to the project.
